#  💳 FinTech: Motor de Reglas de Fraude

## Desafio: Evaluar transacciones entrantes. Si el monto > X o la ubicación es inusual, marcar como sospechosa.


## Visión del Producto

Desarrollar un sistema antifraude que permita detectar transacciones sospechosas de acuerdo al monto y ubicación.  

El sistema analizará cada transacción entrante utilizando reglas definidas por el negocio, Al ubicar montos superiores a un umbral o ubicaciones inusuales, se conseguirá aumentar la seguridad de las transacciones y reducir el riego de fraudes, garantizando una mejor experiencia del usuario.
